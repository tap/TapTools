# Captured primary sources

Drop recovered material here — pages fetched on an unrestricted network (per
`../10-source-checklist.md`), interview answers (per `../11-interview-guide.md`),
or any old artifacts (installers, screenshots, emails) from the Silicon Prairie /
Electrotap / 74 Objects years.

Suggested convention: one file per source, named like `cdm-taptools2-2005.md` or
`interview-act2-norway.md`, with a one-line provenance header (original URL +
capture date, or "memory, <date>"). Once material lands here, the `03`–`09`
dossiers and `00-overview` can be re-synthesized with page-confirmed citations
replacing the `[snippet — verify]` tags.
